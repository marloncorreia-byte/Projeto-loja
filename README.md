# Gestão Simples de Venda e Stock

Projeto de um sistema simples para ajudar pequenos comerciantes a controlar vendas e estoque.

## Problema
Muitos donos de loja anotam tudo no papel ou de cabeça. Isso acaba gerando erros, perda de informação e dificuldade para saber quanto realmente estão ganhando no dia.

## Objetivo
Criar um sistema básico onde seja possível registrar vendas na hora, controlar o estoque e ver o total vendido no dia.

## Público-Alvo
- Donos de pequenas lojas  
- Mercearias  
- Comércio de bairro  

## Funcionalidades (MVP)
- Cadastro de produtos (nome, preço e quantidade)  
- Registro de vendas  
- Atualização automática do estoque  
- Resumo do valor total vendido no dia
